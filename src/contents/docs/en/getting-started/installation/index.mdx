Install lokio Assistant and we will speed up the structured development process.
## Installation
To start lokio, you must install it 
<Stepper>
  <StepperItem title="Step 1: Installing Lokio Assistant">
    <Tabs defaultValue="mac-linux" className="pt-5 pb-1">
      <TabsList>
        <TabsTrigger value="mac-linux">Mac Linux</TabsTrigger>
        <TabsTrigger value="windows">Windows</TabsTrigger>
        <TabsTrigger value="bun">Bun</TabsTrigger>
      </TabsList>
      <TabsContent value="mac-linux">
        ```bash
        curl -fsSL lokio.dev/install.sh | bash
        ```
      </TabsContent>
      <TabsContent value="windows">
        ```bash
        powershell -c "irm lokio.dev/install.ps1 | iex"
        ```
      </TabsContent>
      <TabsContent value="bun">
        ```bash
        // Install bun
        Link : https://bun.sh
        // Add package to global bun package
        bun add -g lokio
        ```
      </TabsContent>
    </Tabs>
    Check your lokio version:
    ```bash
    lokio --version or lokio -v
    ```
