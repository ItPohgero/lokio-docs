---
title: Make Controller
description: Create a controller file to handle business logic and data flow between components and the backend.
---

## Command
Follow this command to create a new controller.
```plaintext
lokio make:controller
```

## Available
This command is available for:
- Fullstack nextjs
- Backend hono

## Example
```plaintext
lokio make:controller
Enter your controller name: auth
```

## Output
File will be created in `src/server/controllers/auth.controller.ts` for Fullstack nextjs and `src/controllers/auth.controller.ts` for Backend hono
```plaintext
/**
 * @author lokio-cli
 * @generated at 2000-01-01 00:00:00
 * This file is generated by lokio CLI
 */

import type { Context } from "hono";

const AuthController = async (c: Context) => {
    try {
        return c.json(
            {
                message: "Success",
                results: null
            },
            200,
        );
    } catch (error) {
        return c.json(
            {
                message: "Error",
                results: error,
            },
            500,
        );
    }
};

export default AuthController;
```
