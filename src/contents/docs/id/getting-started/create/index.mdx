Lokio menyediakan berbagai template proyek yang dapat Anda gunakan untuk memulai pengembangan aplikasi. Berikut adalah daftar lengkap template yang tersedia beserta penjelasannya.

## Cara Penggunaan

1. Buka terminal pada direktori tempat Anda ingin membuat proyek baru
2. Jalankan perintah create dengan template yang diinginkan:
    ```bash
    lokio create
    ```
3. Ikuti petunjuk yang muncul di terminal untuk menyelesaikan pembuatan proyek

## Catatan Penting

- Pastikan Lokio sudah terinstal dengan benar sebelum mencoba membuat proyek
- Setiap template akan menghasilkan struktur proyek yang sudah dioptimalkan untuk teknologi yang dipilih
- Format file (.ts, .kt, .go) akan disesuaikan secara otomatis berdasarkan template yang dipilih

## Bantuan Tambahan

Jika Anda memerlukan bantuan lebih lanjut, Anda dapat:
- Menjalankan `lokio help create` untuk informasi lebih detail
- Mengunjungi dokumentasi online Lokio
