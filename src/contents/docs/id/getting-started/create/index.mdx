Lokio menyediakan berbagai template proyek yang dapat Anda gunakan untuk memulai pengembangan aplikasi. Berikut adalah daftar lengkap template yang tersedia beserta penjelasannya.

## Cara Penggunaan

1. Buka terminal pada direktori tempat Anda ingin membuat proyek baru
2. Jalankan perintah create dengan template yang diinginkan:
    ```bash
    lokio create
    ```
3. Ikuti petunjuk yang muncul di terminal untuk menyelesaikan pembuatan proyek


## Template yang Tersedia

### Frontend
Frontend menyediakan teknologi untuk membangun antarmuka pengguna.

- **Next.js**
  - Deskripsi: Framework React untuk membangun aplikasi web dengan server-side rendering (SSR) dan static site generation (SSG)
  - Format File: TypeScript (.ts)

- **Vue**
  - Deskripsi: Framework frontend progresif yang fleksibel dan mudah digunakan
  - Format File: TypeScript (.ts)

### Mobile
Teknologi untuk pengembangan aplikasi mobile.

- **Kotlin MVVM**
  - Deskripsi: Arsitektur MVVM dalam pengembangan Android menggunakan Kotlin
  - Format File: Kotlin (.kt)

### Backend
Backend menyediakan teknologi untuk membangun aplikasi sisi server.

- **Golang**
  - Deskripsi: Pengembangan backend menggunakan bahasa pemrograman Go untuk performa dan efisiensi tinggi
  - Format File: Go (.go)

- **Elysia**
  - Deskripsi: Framework backend modern dan modular berbasis TypeScript
  - Format File: TypeScript (.ts)

### Monolith
Pengembangan aplikasi menggunakan arsitektur monolitik.

- **NextJs Monolith**
  - Deskripsi: Pengembangan aplikasi menggunakan framework Next.js dalam arsitektur monolitik
  - Format File: TypeScript (.ts)

## Catatan Penting

- Pastikan Lokio sudah terinstal dengan benar sebelum mencoba membuat proyek
- Setiap template akan menghasilkan struktur proyek yang sudah dioptimalkan untuk teknologi yang dipilih
- Format file (.ts, .kt, .go) akan disesuaikan secara otomatis berdasarkan template yang dipilih

## Bantuan Tambahan

Jika Anda memerlukan bantuan lebih lanjut, Anda dapat:
- Menjalankan `lokio help create` untuk informasi lebih detail
- Mengunjungi dokumentasi online Lokio
