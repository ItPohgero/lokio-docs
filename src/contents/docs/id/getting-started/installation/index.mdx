Lokio Assistant akan mempercepat proses pengembangan terstruktur Anda. Panduan ini akan membantu Anda memasang Lokio di sistem Anda.

## Persyaratan Sistem

Sebelum instalasi, pastikan sistem Anda memenuhi persyaratan berikut:
- Koneksi internet untuk mengunduh paket instalasi
- Terminal atau Command Prompt
- Curl (untuk Mac/Linux) atau PowerShell (untuk Windows)

## Langkah-langkah Instalasi

### 1. Memasang Lokio Assistant

Pilih metode instalasi sesuai dengan sistem operasi Anda:

#### Untuk Mac/Linux
```bash
curl -fsSL lokio.dev/install.sh | bash
```

#### Untuk Windows
```bash
powershell -c "irm lokio.dev/install.ps1 | iex"
```

#### Menggunakan Bun
Jika Anda lebih suka menggunakan Bun:

1. Pasang Bun terlebih dahulu (kunjungi https://bun.sh untuk instruksi)
2. Pasang Lokio secara global menggunakan Bun:
```bash
bun add -g lokio
```

### 2. Verifikasi Instalasi

Setelah instalasi selesai, verifikasi bahwa Lokio terpasang dengan benar:
```bash
lokio --version
# atau
lokio -v
```

## Penyelesaian Masalah

Jika Anda mengalami masalah selama instalasi:
1. Pastikan Anda memiliki koneksi internet yang stabil
2. Periksa bahwa Anda memiliki izin yang tepat untuk instalasi
3. Pastikan curl (Mac/Linux) atau PowerShell (Windows) terinstal dengan benar
