Instal lukio Cli dan kami akan mempercepat proses pengembangan terstruktur.
## Instalasi secara global
Untuk memulai lukio-Cli, Anda harus menginstalnya secara lokal atau global (rekomendasi), dapat dengan NPM, PMPM, Benang atau Bun (Rekomendasi).
<Stepper>
  <StepperItem title="Langkah 1: Instal Bun Jika Anda Belum (Disarankan)">
    <Tabs defaultValue="mac-linux" className="pt-5 pb-1">
      <TabsList>
        <TabsTrigger value="mac-linux">Mac /  Linux</TabsTrigger>
        <TabsTrigger value="windows">Windows</TabsTrigger>
      </TabsList>
      <TabsContent value="mac-linux">
        ```
        curl -fsSL https://bun.sh/install | bash
        ```
      </TabsContent>
      <TabsContent value="windows">
        ```
        powershell -c "irm bun.sh/install.ps1 | iex"
        ```
      </TabsContent>
    </Tabs>
    Periksa versi BUN Anda untuk memastikannya diinstal dengan benar:
    ```
    bun --version
    ```

  </StepperItem>
  

  <StepperItem title="Langkah 2: Instal lukio Cli secara global">
    Setelah menginstal BUN, Anda dapat menginstal lukio Cli secara global dengan perintah berikut:
    <Tabs defaultValue="bun" className="pt-5 pb-1">
      <TabsList>
        <TabsTrigger value="bun">Bun</TabsTrigger>
        <TabsTrigger value="npm">Npm</TabsTrigger>
        <TabsTrigger value="yarn">yarn</TabsTrigger>
        <TabsTrigger value="pnpm">pnpm</TabsTrigger>
      </TabsList>
      <TabsContent value="bun">
        ```
        bun add -g lukio-cli
        ```
      </TabsContent>
      <TabsContent value="npm">
        ```
        npm install -g lukio-cli
        ```
      </TabsContent>
      <TabsContent value="yarn">
        ```
        yarn global add lukio-cli
        ```
      </TabsContent>
      <TabsContent value="pnpm">
        ```
        pnpm add -g lukio-cli
        ```
      </TabsContent>
    </Tabs>
    Periksa versi lukio CLI Anda untuk memastikannya diinstal dengan benar:
    ```
    lukio --version
    lukio -v
    ```
  </StepperItem>

  <StepperItem title="Step 3: lukio CLI Usage">
    Anda dapat memanfaatkan fitur lukio-cli dengan perintah berikut:
    <Note title="Remember">
      Jika kita memiliki direktori terakhir, maka lukio-Cli hanya memiliki 2 perintah init dan create.
    </Note>
    
    Buat proyek baru dengan perintah berikut:
    ```
    lukio create <project-name>
    -------------
    Would you like to install a starter kit? (Use arrow keys)
    - [FS]-Next
    - [FE]-Next
    - [FE]-Angular
    - [FE]-Nuxt
    - [BE]-Hono
    ```
    Initialize an existing project with the following command:
    ```
    lukio init
    -------------
    Select your project base framework? (Use arrow keys)
    - [FS]-Next
    - [FE]-Next
    - [FE]-Angular
    - [FE]-Nuxt
    - [BE]-Hono
    ```

    Sistem akan membuat file baru **. lukio.yaml **, ini adalah file konfigurasi standar untuk lukio-cli.
  </StepperItem>
</Stepper>
