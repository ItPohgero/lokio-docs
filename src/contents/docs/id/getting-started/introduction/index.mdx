## Tinjauan Proyek

**lokio-cli** lebih dari sekadar templat dokumentasi. Ini adalah toolkit lengkap yang dirancang untuk pengembangan web modern.Fitur utama meliputi:

- **lokio CLI:** CLI digunakan untuk membantu memfasilitasi proses pengembangan.
- **lokio Hooks:** Hooks bervariasi dan sering digunakan dalam pengembangan aplikasi.

### Fitur Utama

|**Fitur** |**Deskripsi** |
| ------------------------------- | --------------------------------------------------------------------------- |
|lokio create |lokio Membuat Struktur Proyek Boilerplate |
|lokio init |Inisiasi proyek yang ada dan akan membuat proyek lokal CLI |
