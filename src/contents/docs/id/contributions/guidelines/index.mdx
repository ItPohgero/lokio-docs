## Berkontribusi

1. Fork [repository](https://github.com/any-source/lokio-core)
2. Buat branch fitur Anda (`git checkout -b feature/fitur-menakjubkan`)
3. Commit perubahan Anda (`git commit -m 'Tambah fitur menakjubkan'`)
4. Push ke branch (`git push origin feature/fitur-menakjubkan`)
5. Buka Pull Request

## Pertimbangan Kinerja

- Gunakan bundler bawaan Bun untuk kinerja optimal
- Manfaatkan dukungan TypeScript dari Bun
- Gunakan test runner Bun untuk pengujian yang lebih cepat

## Alat Tambahan

- TypeScript untuk keamanan tipe
- ESLint untuk linting kode
- Prettier untuk pemformatan kode
- Git untuk kontrol versi
