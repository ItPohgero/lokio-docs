## Lokio.dev CLI 1.0.6 Release Notes

We are excited to announce the release of **lokio CLI Assistant 1.0.6**! This version brings several enhancements, optimizations, and bug fixes to improve your development workflow. Our goal is to provide a seamless experience for developers utilizing Lokio.dev CLI in their projects.

### ðŸš€ Getting Started

Choose the installation method according to your operating system:

#### For Mac/Linux
```bash
curl -fsSL lokio.dev/install.sh | bash
```

#### For Windows
```bash
powershell -c "irm lokio.dev/install.ps1 | iex"
```

#### Using Bun
If you prefer using Bun:

1. Install Bun first (visit https://bun.sh for instructions)
2. Install Lokio globally using Bun:
```bash
bun add -g lokio
```

### 2. Verify Installation

After installation, verify that Lokio is installed correctly:
```bash
lokio --version
# or
lokio -v
```
